* https://www.e-bebek.com/ sitesi açılır
* Arama çubuğuna "kaşık maması" yazılır ve arama sonucunda gelen sonuçlardan en son ürün açılır, açılan ürünle arama sonucunda tıklanan ürünün aynı olduğu doğrulanır
